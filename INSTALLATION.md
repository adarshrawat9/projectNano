# 🚀 AI Visualizer Pro - Installation & Setup Guide

## Quick Start (5 Minutes)

### Step 1: Prerequisites
- ✅ Chrome browser version 120+ with AI features enabled
- ✅ Gemini Nano model downloaded and ready
- ✅ Developer mode enabled in Chrome

### Step 2: Install Extension
1. Download the `projectNano` folder
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked" and select the `projectNano` folder
5. Pin the extension to your toolbar

### Step 3: Verify AI Availability
1. Click the AI Visualizer Pro icon
2. Check that Gemini Nano is available
3. If not ready, wait for the model to download

### Step 4: Start Using
- **Screen Capture**: Navigate to any webpage and click "Analyze Page"
- **Manual Input**: Switch to manual mode and type your prompts
